# Templates de Paginas Validadas — LTA (Low Ticket Automatizado)

Templates de paginas de venda testadas e validadas em operacoes reais de low ticket. Cada template ja foi usado em campanhas com trafego pago e conversao comprovada.

## Como usar

1. Escolha o template que mais combina com seu nicho
2. Abra o `index.html` e edite textos, imagens e cores
3. Substitua os placeholders de pixel (`SEU_PIXEL_ID`, `SEU_UTMIFY_ID`) pelos seus
4. Suba no seu servidor (Hostinger, Vercel, etc)

| `reference/` | Modelo de referencia | Template master de pagina de vendas |

## Estrutura padrao de uma pagina validada

1. Hero com headline forte + CTA
2. Prova social (depoimentos, numeros)
3. Beneficios / O que voce recebe
4. Garantia + urgencia
5. CTA final com checkout

## Pixels

Cada template tem placeholders para:
- **Meta Pixel** (Facebook/Instagram Ads)
- **Utmify** (rastreamento de UTMs)

Substitua pelos seus IDs antes de publicar.
